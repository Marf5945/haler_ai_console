# Disclaimer

AI Console is an open-source local orchestration layer for AI CLI tools.
It is provided "AS IS" without warranty of any kind.

## Important Notes

1. **Not Medical/Legal/Financial Advice**: AI Console orchestrates AI CLI
   outputs but does not validate their correctness. Never rely on AI-generated
   content for medical, legal, or financial decisions without professional
   consultation.

2. **Local-Only by Design**: AI Console runs entirely on your local machine.
   However, the AI CLI tools it orchestrates (e.g., Claude CLI) may send data
   to external APIs. Review each tool's privacy policy before use.

3. **USB Portability**: While designed for USB portability, you are responsible
   for securing the physical device. Lost or stolen USB drives may expose
   locally stored data.

4. **No Guarantee of Security**: The security mechanisms (Source Trust,
   GatewaySentinel, LLM Context Governance) reduce risk but cannot guarantee
   absolute protection against all attack vectors.

5. **Spec Compliance**: The specPatchChecker validates code against the
   specification but does not constitute a formal security audit.
